# Responsive Strategy

## Breakpoints
- **Wide Desktop**: 1440-1600px. 12 columns, 64px gutter.
- **Standard Desktop**: 1180-1439px. 12 columns, 48px gutter.
- **Tablet / Compact**: 768-1179px. 8 columns, 28-36px gutter. Navigation begins collapsing.
- **Mobile**: 320-767px. 4 columns, 20px gutter (16px minimum). No horizontal overflow.
